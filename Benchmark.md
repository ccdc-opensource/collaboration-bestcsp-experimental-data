# Benchmark

## Introduction

Results from the experiments run on the benchmark compounds will be compiled here. Values labelled with (VT) have variable temperature data recorded. the number in brackets after each value is the
number of experiments collected, hyperlinks will be inserted to access the data where possible. Crystal Parameters, IR and NMR data will be indicated with a hyperlink, since the data won't be scalar.
